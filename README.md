# GO-TAs Corporate

Initial scaffold for the GO-TAs corporate website.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock CMS adapter with future Notion integration
- GitHub Actions CI

## Scripts

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
npm run check
```
