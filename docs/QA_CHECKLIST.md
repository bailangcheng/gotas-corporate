# QA Checklist

- All sitemap routes return a valid page.
- Header and footer links resolve.
- Mobile layout does not overflow at 375px width.
- Text does not overlap fixed UI elements.
- `npm run lint` passes.
- `npm run typecheck` passes.
- `npm run build` passes.
- Magazine list and detail pages work with mock data.
- Contact page clearly indicates final form provider is pending.
- No server secrets are exposed to client code.
