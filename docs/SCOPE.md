# Scope

## In scope for initial build

- Next.js project foundation.
- Sitemap-based routing.
- Common layout, header, footer, section components, and global CSS tokens.
- Dummy lower pages based on the current sitemap.
- Magazine/news list and detail pages using a mock CMS adapter.
- Notion CMS adapter boundary and schema plan.
- Contact page placeholder and implementation policy.
- GitHub CI for lint, typecheck, and build.
- Vercel-oriented deployment preparation.

## Out of scope unless separately agreed

- Pixel-perfect lower page implementation before wireframes are final.
- Full custom CMS or admin dashboard.
- Firebase/Supabase backend.
- Production DNS ownership or domain contract.
- Hosting account ownership.
- Long-term maintenance and major post-release revisions.
- Legal copy creation, privacy policy drafting, or compliance review.
- Large-scale animation, video production, or image retouching.
