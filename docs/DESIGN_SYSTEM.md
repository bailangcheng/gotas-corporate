# Design System

## Current status

This is a working design foundation before all Figma lower pages are fixed.

## Typography

- Primary font stack: `Noto Sans JP`, `Hiragino Sans`, `Yu Gothic`, `Meiryo`, sans-serif.
- Do not scale type by viewport width.
- Use tight but readable corporate typography.

## Color tokens

- Brand blue: `#1d4ed8`
- Dark blue: `#1e3a8a`
- Slate text: `#0f172a`
- Muted surface: `#f8fafc`
- Subtle border: `#e2e8f0`

## Layout

- Max content width: `72rem`
- Page sections should use full-width bands, not nested decorative cards.
- Cards are reserved for repeated items and framed tools.

## Component policy

- Build common primitives first:
  - Container
  - ButtonLink
  - SectionHeading
  - PageHero
  - Header/Footer
- Page-specific content should compose shared primitives.
